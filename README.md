# xemX Materials Homepage

Static single-page homepage for xemX materials space exploration GmbH.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- lucide-react
- framer-motion

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Structure

```txt
src/components  Reusable page sections and visual components
src/data        Editable homepage copy and card content
src/lib         Small utilities
```

The contact form is static for now. Wire it to a form provider or backend when the submission target is known.
