xemX USE CASES AND EXPERIMENTAL CAPABILITIES
Authoritative guide for site and proposal editing. This is source material,
not finished public copy.

Purpose
This guide defines what the xemX website may say, what should stay on detail
pages, and what still needs project-specific wording. It uses the local xemX
repo as the content authority, then checks that surface against the literature
from Lars Banko, Alfred Ludwig, Wolfgang Schuhmann, Ruhr University Bochum, and
ZGH-related work.

Authority order
1. Local xemX files in this repo.
2. Literature with direct xemX, Banko, Ludwig, or Schuhmann involvement.
3. General Ruhr University Bochum and ZGH capability evidence.
4. Public company pages only as obsolete background, since this repo is
   replacing that surface.

Core positioning
xemX runs measured materials screening campaigns.

The core job:
- Create physical thin-film material libraries where composition changes
  across a wafer or sample set.
- Measure many real material variants.
- Return composition-property maps.
- Help R&D teams choose smaller validation sets.

Public wording should stay close to the work:
- real samples
- thin-film libraries
- composition gradients
- measured points
- local electrochemical screening
- XRD, EDX / WDX, electrical, optical, magnetic, and mechanical maps
- candidate regions
- final validation after screening

Avoid these frames:
- generic AI materials company
- generic coatings shop
- final manufacturing approval provider
- full device validation provider
- full battery cell validation provider
- full semiconductor integration provider
- production qualification provider


1. LITERATURE BASIS

1.1 Direct xemX / Banko / Ludwig / Schuhmann evidence

Combinatorial thin-film libraries and measured maps
- Literature supports co-sputtered thin-film material libraries, 100 mm wafer
  formats, 342 measurement areas, EDX composition maps, XRD maps, and
  four-point probe resistance maps.
- Use publicly: yes.
- Website translation: xemX creates thin-film libraries, measures many points,
  and returns composition-property maps.
- Key source: Thelen, Banko, Zehl, Baha, Ludwig, Digital Discovery 2023,
  DOI 10.1039/D3DD00125C.

High-entropy alloy and complex-solid-solution electrocatalysts
- Literature supports HEA and complex-solid-solution electrocatalyst discovery
  using co-sputtered libraries and high-throughput electrochemical screening.
- Use publicly: yes.
- Website translation: xemX can test multicomponent catalyst libraries before
  customer protocol, electrode, cell, or stack validation.
- Key sources:
  - Banko et al., deposition source permutations, arXiv:2106.08776.
  - Banko et al., Advanced Energy Materials 2022, DOI 10.1002/aenm.202103312.
  - Batchelor et al., Angewandte Chemie 2021, DOI 10.1002/anie.202014374.

Water electrolysis and HER / OER / ORR screening
- Literature supports HER, OER, and ORR screening on catalyst libraries.
- Use publicly: yes.
- Website translation: xemX can map local activity and stability-relevant
  behavior across catalyst compositions, then select regions for validation.
- Key sources:
  - Mo-Ru HER work, DOI 10.1002/celc.202400631.
  - Ni-Pd-Pt-Ru OER Bayesian optimization work, DOI 10.1002/advs.202507302.
  - Co-Fe-Ni OER screening work, DOI 10.1002/cctc.202401749.

Scanning Droplet Cell and SECCM
- Literature supports local electrochemical screening through Scanning Droplet
  Cell and SECCM-style methods.
- Use publicly: yes.
- Website translation: SDC is a core local electrochemical screening method.
  SECCM can be described as a project-dependent higher-resolution route.
- Key sources:
  - Schuhmann group micro/nanoelectrochemistry pages and papers.
  - Banko et al., Advanced Materials 2023, DOI 10.1002/adma.202207635.

Data-guided measurement and follow-up
- Literature supports active learning, Gaussian-process measurement selection,
  and Bayesian optimization in materials-library workflows.
- Use publicly: yes, translated into plain language.
- Website translation: measured maps can guide the next measurement, the next
  library, or a smaller validation set.
- Avoid public headline: "AI materials platform."
- Key sources:
  - Thelen, Banko, Zehl, Baha, Ludwig, Digital Discovery 2023,
    DOI 10.1039/D3DD00125C.
  - Thelen et al., Advanced Science 2025, DOI 10.1002/advs.202507302.

XRD interpretation support
- Literature supports XCA-style automated XRD phase identification and VAE-style
  novelty detection for large XRD data sets.
- Use publicly: detail page or methods page only.
- Website translation: software can help interpret large XRD maps; expert review
  and measured data remain central.
- Key source: Banko, Maffettone, Naujoks, Olds, Ludwig, npj Computational
  Materials 2021, DOI 10.1038/s41524-021-00575-9.

Oxide and photoelectrochemical thin-film libraries
- Literature supports oxide libraries and photoelectrochemical screening,
  including Ti-W-O, Fe-W-O, Fe-V-O, Ag-V-O, Bi-Mo-W-O, and related materials.
- Use publicly: yes.
- Website translation: xemX can map oxide and nitride thin-film regions where
  structure, stoichiometry, optical response, electrical response, or
  photoelectrochemical behavior matter.
- Key sources:
  - ChemSusChem 2015 semiconductor material library papers,
    DOI 10.1002/cssc.201402917 and DOI 10.1002/cssc.201402918.
  - Fe-V-O solar water splitting, DOI 10.1021/acscombsci.8b00030.
  - Bi-Mo-W-O photoelectrochemical library, DOI 10.1515/zpch-2019-1439.

Reactive sputtered nitride and hard-film libraries
- Literature supports Cr-Al-N and related reactive sputtered nitride studies
  tied to automated XRD, plasma diagnostics, and mechanical properties.
- Use publicly: yes, as oxide/nitride and hard-film screening.
- Website translation: xemX can screen reactive sputtered nitrides and oxides
  where composition, phase, hardness, and process fit matter.
- Key source: Banko et al., ACS Combinatorial Science 2019,
  DOI 10.1021/acscombsci.9b00123.

Magnetic and shape-memory screening
- Ruhr/ZGH/Ludwig literature supports MOKE mapping, magnetic thin-film
  libraries, and ferromagnetic shape-memory alloy library screening.
- Use publicly: yes, as an application family. Keep final device and stack
  validation outside the claim.
- Website translation: xemX can map magnetic response across thin-film
  composition libraries and identify regions for stack-specific validation.
- Key sources:
  - Ludwig, npj Computational Materials 2019, DOI 10.1038/s41524-019-0205-0.
  - Thienhaus, Hamann, Ludwig, Science and Technology of Advanced Materials
    2011, DOI 10.1088/1468-6996/12/5/054206.

Follow-up samples and electrode transfer
- Literature supports cases where hit compositions from thin-film screens were
  transferred into powder, electrode, foam, or gas-diffusion-electrode formats.
- Use publicly: yes, with careful wording.
- Website translation: selected regions can guide follow-up samples or customer
  validation formats.
- Avoid promise: standard scale-up, manufacturing transfer, or production
  approval.
- Key sources:
  - Mo-Ru HER work, DOI 10.1002/celc.202400631.
  - Co-Cu-Mo-Pd-Re to gas diffusion electrodes, DOI 10.1002/adfm.202400180.
  - Co-Fe-Ni OER work, DOI 10.1002/cctc.202401749.

CO2 and battery-interface examples
- Literature and Ruhr electrochemistry context support CO2-related
  electrochemistry and battery-interface methods as plausible project-dependent
  questions.
- Use publicly: yes, as project-dependent examples.
- Website translation: use one careful line in electrochemical applications or
  detail pages. Do not imply full battery-cell validation or guaranteed CO2
  product selectivity.

Semiconductor, RF acoustic, package, and interface examples
- Literature supports thin-film semiconductor material libraries and general
  thin-film screening methods. The local guide includes semiconductor,
  RF acoustic, package, barrier, liner, cap, contact, and interface examples.
- Use publicly: yes, as application families and campaign examples.
- Website translation: these are screening questions before device, resonator,
  stack, package, or reliability validation.

- Magnetron co-sputtering.
- Reactive sputtering.
- Composition gradients.

- EDX / WDX composition mapping.
- XRD phase and structure mapping.
- Four-point probe resistance or resistivity mapping.
- Scanning Droplet Cell local electrochemical screening.
- UV-VIS optical screening.
- MOKE magnetic screening.
- Nanoindentation for hardness and modulus.
- Measured composition-property maps.
- Candidate-region selection.
- Data-guided follow-up.
- Final validation happens after screening.

2.2 Good detail-page claims

Use on methods, applications, or examples pages:
- SECCM as a higher-resolution electrochemical route.
- Long-range SECCM as project-dependent support.
- XCA-assisted XRD analysis.
- VAE-style XRD novelty detection.
- Bayesian optimization.
- Gaussian-process measurement selection.
- Deposition source permutations.
- Microscale combinatorial libraries.
- Follow-up uniform depositions.
- Electrode, foam, powder, or gas-diffusion-electrode transfer where scoped.
- CO2 electroreduction as a project-dependent electrochemical question.
- Battery interface / SEI screening as a project-dependent electrochemical
  question.
- Semiconductor-relevant thin-film screening.
- RF acoustic film screening.
- Magnetic-library screening.
- Shape-memory and ferromagnetic shape-memory directions.

2.3 Keep vague

Use broad wording until a proposal or approval defines the route:
- Bochum, Ruhr, or ZGH ecosystem support.
- Broader characterization routes.
- Cleanroom support.
- Microfabrication support.
- Etch and patterning support.
- Facility access.
- Partner-supported scale-up.
- Manufacturing transfer support.

Approved vague wording:
- "Project-dependent follow-up routes can connect selected regions to deeper
  characterization or validation support."
- "Where the project requires it, selected candidates can move into additional
  characterization or customer validation formats."
- "The work is rooted in Ruhr University Bochum and ZGH research experience,
  with project scope defined before follow-up work begins."

2.4 Avoid unless approved for a specific proposal

Do not use as default public claims:
- Full device validation.
- Full battery-cell validation.
- Full semiconductor integration.
- Full resonator validation.
- Hardware reliability approval.
- Package-level qualification.
- Production coating scale-up.
- Full manufacturing transfer.
- Production approval.
- Guaranteed facility access.
- Named customer work.
- Named partner work.
- Patent claims.
- Citation-count claims.
- Standard delivery promises.


3. APPLICATION FAMILIES

3.1 Electrocatalyst discovery

Public status: homepage core.

Material questions:
- Which mixed-metal surface gives the activity and stability profile worth
  repeating under the customer test protocol?
- Which catalyst or electrode material region should be repeated or refined?
- Which composition regions show promising local activity or surface response?

Campaign shape:
- Prepare catalyst or electrode material libraries.
- Run localized electrochemical measurements across the library.
- Tie activity, stability, or surface response back to composition and phase.
- Return activity maps and candidate regions.

Examples:
- HER / hydrogen evolution.
- OER / oxygen evolution.
- ORR / oxygen reduction.
- Water electrolysis material directions.
- Mixed-metal catalysts.
- Complex catalyst compositions.
- Catalyst, electrode, and electrochemical surface questions.
- Corrosion-relevant material libraries where local electrochemistry applies.
- CO2 electroreduction, where scoped.
- Battery interface / SEI questions, where scoped.

Boundary:
- Full cell, stack, hardware, or customer protocol validation follows screening.

3.2 Complex alloys and solid solutions

Public status: homepage or applications page.

Material questions:
- Which multicomponent alloy region is worth preparing again for a target
  property?
- Which region balances phase, hardness, electrical behavior, magnetic
  response, activity, or stability?
- Which high-entropy or complex-solid-solution direction should move to deeper
  validation?

Campaign shape:
- Create composition-spread alloy libraries.
- Map composition, phase, and the property tied to the decision.
- Compare tradeoffs across multicomponent spaces.
- Return candidate regions and regions to avoid.

Examples:
- High-entropy alloys.
- Complex solid solutions.
- Shape-memory alloy directions.
- Ferromagnetic shape-memory alloy directions.
- Multicomponent alloy spaces.

Boundary:
- Bulk alloy, powder, part-level, or final-format validation follows thin-film
  screening.

3.3 Oxide and nitride libraries

Public status: homepage or applications page.

Material questions:
- Which oxide or nitride region forms the desired phase and property
  combination?
- Which stoichiometry or process range gives the required structure or
  functional response?
- Which reactive sputtering route should be narrowed for follow-up?

Campaign shape:
- Use reactive sputtering to create oxide or nitride material libraries.
- Map composition, structure, stoichiometry, and functional properties.
- Link phase formation and property response.
- Run a narrower campaign or prepare selected follow-up films.

Examples:
- Perovskite oxides.
- Functional oxides.
- Transition-metal nitrides.
- Electrocatalyst-relevant oxides.
- Functional nitrides.
- Hard films.
- Optical films.
- Semiconductor-adjacent layers.

Boundary:
- Final device, catalyst, coating, or hardware validation follows screening.

3.4 Semiconductor-relevant thin films

Public status: applications page and campaign examples.

Material questions:
- Which W, Mo, Ru, or related thin-film region keeps resistivity low enough at
  the target thickness?
- Which barrier, liner, cap, contact, gate-insulator-adjacent oxide, or memory
  stack material should move into device-level testing?
- Which composition, phase, resistivity, or texture window is worth narrowing?

Campaign shape:
- Create thin-film material libraries around a semiconductor-relevant layer
  question.
- Map phase, texture, resistivity, composition, and structure.
- Return low-resistance or process-relevant candidate windows.

Examples:
- Low-resistance metals.
- Contacts.
- Barrier layers.
- Liners.
- Caps.
- Gate-insulator-adjacent oxides.
- Memory stacks.
- Isolation-adjacent areas.
- Semiconductor-adjacent functional films.

Boundary:
- Full semiconductor integration and device-level testing follow screening.

3.5 RF acoustic films

Public status: applications page and campaign examples.

Material questions:
- Which AlScN, ScAlN, or related composition and process window gives the
  needed texture without unwanted phase formation?
- Which composition windows deserve resonator-level testing?

Campaign shape:
- Create gradient thin-film libraries.
- Compare composition, structure, texture, and electrical or mechanical
  proxies.
- Return candidate composition and process windows.

Examples:
- AlN.
- AlScN.
- ScAlN.
- Texture windows.
- Piezoelectric-response-relevant films.

Boundary:
- Resonator-level testing and customer device-stack validation follow the
  campaign.

3.6 Magnetic thin films

Public status: applications page and campaign examples.

Material questions:
- Which composition regions reach the target magnetic response while keeping
  the needed phase and structure?
- Which magnetic multilayer or reduced-rare-earth direction deserves deeper
  stack-specific validation?

Campaign shape:
- Screen composition, structure, and magnetic behavior across a thin-film
  material library.
- Tie MOKE magnetic response to composition and phase data.
- Return magnetic response maps and candidate regions.

Examples:
- Magnetic multilayers.
- Coercivity screening.
- Magnetic response mapping.
- Reduced-rare-earth directions.
- Ferromagnetic shape-memory directions, if relevant to the project.

Boundary:
- Stack-specific validation follows screening.

3.7 Optical and photoelectrochemical films

Public status: applications page and campaign examples.

Material questions:
- Which film region balances optical response, composition, phase, and
  stability-relevant behavior?
- Which transparent, reflective, absorptive, photonic, or photoresponse region
  deserves follow-up?

Campaign shape:
- Map optical or photoelectrochemical response, composition, and structure
  across a controlled thin-film library.
- Compare reflectance, absorption, transparency, conductivity, stability, or
  photoresponse.
- Return optical-property maps and tradeoff regions.

Examples:
- Transparent films.
- Reflectance-tuned films.
- Absorption-tuned films.
- Photonic thin films.
- Photoelectrochemical films.
- Functional optical coatings.

Boundary:
- Final coating stack, photonic device, or photoelectrochemical validation
  follows screening.

3.8 Protective coatings and packaging interfaces

Public status: applications page and campaign examples.

Material questions:
- Which coating region keeps conductivity while improving durability or
  chemical stability?
- Which bonding, barrier, adhesion, current-collector, protective, or package
  interface material should move into reliability testing?

Campaign shape:
- Screen composition, structure, resistance, mechanical response, and
  interface-relevant behavior.
- Return tradeoff maps showing where performance starts to fail.
- Build smaller validation sets around the best tradeoff regions.

Examples:
- Conductive protective surfaces.
- Bonding layers.
- Diffusion barriers.
- Adhesion layers.
- Current collector films.
- Package-adjacent films.
- Interface materials.

Boundary:
- Hardware or package-level reliability testing follows screening.

  behavior across compositions.

4.3 Choosing the next experiment

Core:
- Campaign design around the material question.
- Composition-property maps.
- Multi-property tradeoff comparison.
- Candidate-region selection.
- Regions-to-avoid identification.
- Measured-map guided follow-up.
- Data-guided measurement selection.
- Data-guided follow-up campaigns.
- Thin-film proxy discipline.

Project-dependent:
- Bayesian optimization.
- Gaussian-process measurement selection.
- Automated pattern analysis.
- Automated XRD interpretation support.
- XCA-assisted XRD analysis.
- VAE-style XRD novelty detection.
- Modeling and data analysis support for campaign design and interpretation.

Plain meaning:
- The campaign defines the material space, the measurements, and the decision.
- The first measured map can guide the next measurement, sample, or campaign.
- Software can support interpretation. The core output is measured data from
  real samples.

7. ONE-LINE USE-CASE TEMPLATE

Use this template for new use cases:

We test [real material variants] to measure [property or failure mode] before
[expensive validation step].

Examples:
- We test AlScN composition windows to measure texture and phase behavior
  before resonator-level validation.
- We test mixed-metal catalyst libraries to measure local activity and
  stability before full-cell or customer-protocol validation.
- We test contact or barrier thin-film regions to measure resistivity,
  composition, and phase before device-level testing.
- We test coating libraries to measure resistance, mechanical response, and
  stability before hardware reliability testing.
- We test oxide or nitride libraries to measure phase, stoichiometry, and
  functional response before selected-film validation.
