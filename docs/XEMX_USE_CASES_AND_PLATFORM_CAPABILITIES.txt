XEMX USE CASES AND PLATFORM CAPABILITIES
Working inventory for review. This is not automatically public-approved copy.

Purpose
This document collects the broad possible use-case surface and the platform
specification/capability surface from the current local XemX materials site
source. It is written as a practical inventory: what the platform can talk
about, what a campaign can do, and what needs approval before public launch.

Core positioning
xemX is best understood as an experimental materials search company.
The core job is:
- Create physical material libraries where composition changes across a wafer
  or sample set.
- Measure many real material variants.
- Return measured composition-property maps.
- Help R&D teams decide which material regions deserve expensive validation.

Important boundary
xemX should not be framed as a generic AI materials company, a generic coatings
shop, or a full manufacturing/final-validation provider. The campaign narrows
candidate regions. Device, cell, stack, hardware, package, part-level, and
production validation happen after screening.


1. USE CASE FAMILIES

1.1 Electrocatalyst discovery
Potential material questions:
- Which mixed-metal or complex-solid-solution surface gives useful activity
  and stability for the chosen reaction?
- Which catalyst or electrode material region should be repeated or refined?
- Which composition regions show promising local activity or surface response?

Likely campaign shape:
- Prepare catalyst or electrode material libraries.
- Run localized electrochemical measurements across the library.
- Tie activity, stability, or surface response back to composition and phase.
- Return an activity/stability map and candidate regions.

Examples and reaction areas:
- HER / hydrogen evolution reaction.
- OER / oxygen evolution reaction.
- ORR / oxygen reduction reaction.
- Water electrolysis material directions.
- Mixed-metal catalysts.
- Complex catalyst compositions.
- Catalyst, electrode, and electrochemical surface questions.
- Corrosion-relevant material libraries where local electrochemistry applies.

Boundary:
- Full cell, stack, hardware, or customer protocol validation comes after the
  screening campaign.
- CO2 electroreduction and battery SEI/battery-interface screening are
  potential/approval-needed claims before public use.


1.2 Complex alloys and solid solutions
Potential material questions:
- Which multicomponent alloy region is worth preparing again for a target
  property?
- Which composition region balances phase, hardness, electrical behavior,
  magnetic response, activity, or stability?
- Which high-entropy or complex-solid-solution direction should be selected
  for deeper validation?

Likely campaign shape:
- Create composition-spread alloy libraries.
- Map composition, phase, and the property tied to the decision.
- Compare tradeoffs across multicomponent spaces.
- Return candidate regions and regions to avoid.

Examples:
- High-entropy alloys.
- Complex solid solutions.
- Shape-memory alloy directions.
- Ferromagnetic shape-memory alloy directions.
- Multicomponent alloy spaces.

Boundary:
- Bulk alloy, powder, part-level, or final-format validation comes after the
  thin-film screening campaign.
- Shape-memory and ferromagnetic shape-memory public claims should be checked
  against current commercial scope before launch.


1.3 Oxide and nitride libraries
Potential material questions:
- Which oxide or nitride composition region forms the desired phase and
  property combination?
- Which stoichiometry or process range gives the required structure or
  functional response?
- Which reactive sputtering route should be narrowed for follow-up?

Likely campaign shape:
- Use reactive sputtering to create oxide or nitride material libraries.
- Map composition, structure, stoichiometry, and functional properties.
- Link phase formation and property response.
- Run a narrower campaign or prepare selected uniform films.

Examples:
- Perovskite oxides.
- Functional oxides.
- Transition-metal nitrides.
- Electrocatalyst-relevant oxides.
- Functional nitrides.
- Hard films.
- Optical films.
- Semiconductor-adjacent layers.

Boundary:
- Final device, catalyst, coating, or hardware validation comes after the
  screen.


1.4 Semiconductor thin films
Potential material questions:
- Which W, Mo, Ru, or related thin-film region gives useful electrical
  resistance at thin-film thickness?
- Which barrier, liner, cap, contact, gate-insulator-adjacent oxide, or memory
  stack material should move into device-level testing?
- Which composition, phase, resistivity, or texture window is worth narrowing?

Likely campaign shape:
- Create thin-film material libraries around a semiconductor-relevant layer
  question.
- Map phase, texture, resistivity, composition, and structure.
- Return low-resistance or process-relevant candidate windows.

Examples:
- Low-resistance metals.
- Contacts.
- Barrier layers.
- Liners.
- Caps.
- Gate-insulator-adjacent oxides.
- Memory stacks.
- Isolation-adjacent areas.
- Semiconductor-adjacent functional films.

Boundary:
- Full semiconductor integration and device-level testing come after screening.
- Public semiconductor wording should avoid implying fab-qualified final
  integration unless explicitly approved.


1.5 RF acoustic films
Potential material questions:
- Which AlScN, ScAlN, or related composition and process window gives the
  needed texture without unwanted phase formation?
- Which composition windows deserve resonator-level testing?

Likely campaign shape:
- Create gradient thin-film libraries.
- Compare composition, structure, texture, and electrical or mechanical
  proxies.
- Return candidate composition/process windows.

Examples:
- AlN.
- AlScN.
- ScAlN.
- Texture windows.
- Piezoelectric-response-relevant films.

Boundary:
- Resonator-level testing and customer device-stack validation come after the
  campaign.


1.6 Magnetic thin films

Likely campaign shape:

- Tie MOKE magnetic response to composition and phase data.
- Return magnetic response maps and candidate regions.

Examples:

- Coercivity screening.
- Magnetic response mapping.
- Reduced-rare-earth directions.

Boundary:

Likely campaign shape:

- Return optical-property maps and tradeoff regions.

Examples:
- Transparent films.
- Reflectance-tuned films.
- Absorption-tuned films.
- Photonic thin films.
- Photoelectrochemical films.
- Functional optical coatings.

Boundary:

1.8 Protective coatings and packaging interfaces
Potential material questions:
- Which coating region keeps conductivity while improving durability or

- Which bonding, barrier, adhesion, current-collector, protective, or package

Likely campaign shape:
- Screen composition, structure, resistance, mechanical response, and
  interface-relevant behavior.
- Return tradeoff maps showing where performance starts to fail.
- Build smaller validation sets around the best tradeoff regions.

Examples:
- Conductive protective surfaces.
- Bonding layers.
- Diffusion barriers.
- Adhesion layers.
- Current collector films.
- Package-adjacent films.
- Interface materials.

Boundary:
- Hardware or package-level reliability testing comes after screening.


1.9 Broader cross-cutting campaign use cases
These are not separate industries as much as reusable decision patterns:
- Too many candidate compositions to test one by one.
- Expensive final validation should not be the first filter.
- Several properties matter at once.
- A material works for one property but fails on phase, texture, conductivity,
  stability, process fit, hardness, optical response, magnetic behavior, or
  electrochemical response.
- Literature or simulation narrows ideas but cannot replace measured data from
  real samples.
- The customer needs a measured map before choosing a smaller validation set.
- A first campaign needs to guide a second, tighter campaign.
- A promising region needs follow-on uniform samples.
- A larger multielement search space needs campaign-by-campaign sampling
  without pretending every composition has been covered.


2. CAMPAIGN EXAMPLES

2.1 Low-resistance thin films
Category: semiconductor materials.
Question: which W, Mo, Ru, or related composition region gives useful
electrical resistance at thin-film thickness?
Campaign: map composition, phase, texture, and resistivity across a thin-film
library.
Output: candidate regions for customer-specific process or device validation.
Next step: tighter process windows on selected film families.

2.2 AlScN film screening
Category: RF acoustic films.
Question: which AlScN composition and process window gives the needed texture
without unwanted phase formation?
Campaign: create gradient libraries and compare composition, structure, and
electrical or acoustic proxies.
Output: composition regions that deserve resonator-level testing.
Next step: transfer best windows into customer device stacks.

2.3 Electrocatalyst composition search
Category: localized electrochemistry.
Question: which mixed-metal or complex-solid-solution surface gives useful
activity and stability for the chosen reaction?
Campaign: prepare catalyst material libraries and run localized electrochemical
measurements.
Output: measured activity map with regions to repeat or refine.
Next step: validate selected catalysts under the customer test protocol.

2.4 High-entropy alloy screen
Category: complex alloys.
Question: which multicomponent alloy region is worth preparing again for a
target property or electrochemical response?
Campaign: create a composition-spread alloy library and map composition,
phase, and the property tied to the decision.
Output: composition-property map showing candidate regions and regions to
avoid.
Next step: prepare follow-up samples around the best region for customer
validation.

2.5 Oxide or nitride library
Category: reactive sputtering.
Question: which oxide or nitride composition region forms the desired phase and
property combination?
Campaign: use reactive sputtering to create a library, then map composition,
structure, and the relevant functional property.
Output: measured map linking stoichiometry, phase, and property response.
Next step: run a narrower campaign or prepare selected uniform films.

2.6 Conductive protective coating
Category: coatings and interfaces.
Question: which coating region keeps conductivity while improving durability
or chemical stability?
Campaign: screen composition, structure, resistance, and mechanical response
across a coating library.
Output: tradeoff map showing where performance starts to fail.
Next step: build a smaller validation set around the best tradeoff region.

2.7 Magnetic thin-film screening
Category: magnetic materials.
Question: which composition regions show useful magnetic response while keeping
the needed phase and structure?
Campaign: screen composition, structure, and magnetic behavior across a
thin-film material library.
Output: magnetic response map tied to composition and phase data.
Next step: move selected regions into stack-specific or device-adjacent
validation.

2.8 Optical or photoelectrochemical film
Category: functional thin films.
Question: which film regions balance optical response, composition, phase, and
stability-relevant behavior?
Campaign: map optical or photoelectrochemical response, composition, and
structure across a controlled thin-film library.
Output: optical-property map showing candidate regions and tradeoffs.
Next step: prepare follow-up films for customer coating or photonic test
structures.

Plain meaning:
- Many related material compositions can be created in one campaign instead of
  one sample at a time.

- Each measured location can become a point on a composition-property map.
- Promising regions can be prepared again as simpler or more uniform follow-up
  samples.

6. ONE-LINE USE-CASE TEMPLATE

Use this template for new use cases:

We test [real material variants] to measure [property or failure mode] before
[expensive validation step].

Examples:
- We test AlScN composition windows to measure texture and phase behavior
  before resonator-level validation.
- We test mixed-metal catalyst libraries to measure local activity and
  stability before full-cell or customer-protocol validation.
- We test contact or barrier thin-film regions to measure resistivity,
  composition, and phase before device-level testing.
- We test coating libraries to measure resistance, mechanical response, and
  stability before hardware reliability testing.
